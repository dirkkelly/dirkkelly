## Disqus

You can optionally enable a comment system powered by Disqus for the posts. Simply add the variable `disqusShortname` to the `params` in your config file.

**TOML**
```toml
[params]
  disqusShortname = "dirkkellycom"
```

**YAML**
```yaml
params:
  disqusShortname: "dirkkellycom"
```
